Meet NEET computers!

But what is a neet computer?

NEET is a minecraft mod (fabric modloader) that adds real computers to minecraft that can run real code, a point of comparison an older mod and primary inspiration ComputerCraft or its modern fork CC:Tweaked. where CC and its advanced counterpart might have founded the idea of running code (lua in particular) to minecraft, NEET attempts to improve that groundwork and create a system with vivid graphics, full user io, speedy code execution, and basic user simplicity for players without a programing background to still be able to interact with and enjoy computing in minecraft!

But why a neet computer

Unlike its predecessor, NEET utilised full 24 bit color with 384 x 288, 4/3 GUI graphics and a programmatic display that renders directly on the block model! It also has full filesystem support including binary files, partitions, and live rewritable OS folders. network support is coming soon (hopefully), and peripherals are fully implemented (even if it's a limited selection).
