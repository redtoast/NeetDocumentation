Simple Display Documentation

Introduction

The Simple Display is a large fun multi block graphical display!

When multiple simple displays are placed together, they will merge to form a larger display. From the perspective of a computer They are all one peripheral and will work the same as a projector display, but with a larger screen size.

You can also shift click the display with an empty hand to clear it, unless disabled in mod settings.
Holding shift while placing a display will make it only connect with other displays placed while holding shift.

The Color Display is categorized as "neetcomputers:color_display" in the IO API

The API is the exact same as the Headsup API!
