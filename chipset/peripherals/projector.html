Chipset / Peripherals / 'projector screen'

This peripheral represents the computers filesystem, and is created on the computers runtime

draw()

The draw function takes no arguments and draws the graphics buffer to the screen!

clear()

Wipes the screen

getSize()

Returns Width, and Height, representing the dimensions of the screen

Drawing operations

These operations allow you to draw data to the screen

drawPixel(x, y)

Turns the pixel at x, y on

drawLine(x1, y1, x2, y2)

Activates a line between the two points

drawRec(x1, y1, x2, y2)

Activates a solid color square between the two points
