Chipset / Chip API

The chip API is dedicated to extracting important system information

shutdown()

Forces the entire computer to terminate

crash(Message)

Stops the computers runtime completely without waiting for the tick cycle to end, and enters a crashed state with Message

Getting system data

This section is all about retrieving data on the computer running this thread

getTime()

Gets the current time in Unix time

getTimeAlive()

Gets the amount of milliseconds sense the computer has started ticking

getUUID()

Returns the UUID of this computer as a string, this uuid is persistent

version()

Gets the current running version of N.E.E.T. computers

getMachine()

Returns the name of the model of computer your using

Core management

This section explains how to manage Cores

getCurrentCore()

Returns the UUID of the core executing this command

getCoreCount()

Returns the amount of cores currently running

getMaxCoreCount()

Finds the maximum amount of cores this computer can run

createCore(script)

Opens a new core running the provided script and returns the cores UUID, if the core cap has not been exited, throws if it has

killCore(UUID)

Terminates the core that UUID belongs too, and returns if this operation was successful
