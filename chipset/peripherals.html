Chipset / Peripherals API

The peripheral API gives scripts access to the peripherals available to the computer

exists(UUID)

Given a peripherals UUID, it will determine if that peripheral exists

isPeripheral(table)

Verify that table is a peripheral based of its metadata

getUUID(peripheral)

Gets the UUID of peripheral

getType(peripheral)

Gets the type of peripheral as a string

retrieve(UUID)

Retrieves the peripheral with the UUID, or null if no such peripheral exists

locate(type)

Returns all peripherals with this type in the form of a tuple

getAll()

Returns a list of every peripheral registered
