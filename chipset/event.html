Chipset / Event API

The event API allows programs to access the built-in event system, giving programs access to user input and other useful data!

getEventqueue()

Returns a List of all events in the queue, the queue files with events as they happen, each event is represented with a list containing the event name and then custom data

clearEventQueue()

Manually clears the event queue, make sure to call this function because the queue is not cleared automatically except to make room if it reaches a size of 100

queueEvent(eventName, arguments...)

Manually inject an event into the queue next tick

registerEvent(eventName, function)

Before each tick, every instance of the eventName will be passed as a list into the function, the function will be called with the event name and parameters provided individually

registerEvent(function)

Before each tick, every instance of any event will be passed as a list into the function, unlike registerEvent the event is passed as a list. Also keep in mind the event queue is capped at 100 events and will start forgetting old events if that cap is reached
