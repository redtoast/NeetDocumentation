Capabilities

NEET computers supports a full file system, meaning partitions, rewritable ROM's, binary files, and all the file access modes I could think of!

Computers start off with a partitioned filesystem, the pre-installed files are fully editable both with data packs to determine content and can be edited by the computer itself, unlike in CC/CC:T where the os is read only! (planned) Additional file systems (in the form of hard drives, or boot drives) can be mounted and edited, and drives are able to be finalised with a bootup path and crafted into a computer to act as its file system, for full custom os's, no more wrapper os's

Restrictions?

As cool as it is, the file-system system is not perfect. it's relatively slow (although there's a faster 'flash' API to remedy this) and I haven't added non-partitioned drives don't currently exist (but they might one day)
