Chipset / Screen API

draw()

The draw function takes no arguments and draws the graphics buffer to the screen!

getSize()

Returns Width, and Height, representing the dimensions of the screen

Applying global attributes

This section contains documentation for applying effects to other drawing bills

setColor(R, G, B)

The most basic function for setting the default color of drawn graphics, note that every draw function can have its RGB color overridden by adding the RGB values after the functions original parameters. (I.E. drawPixel(x, y, R, G, B))

setColor(Alpha)

Sets the transparency of drawn graphics, ranging 1 - 256 with 1 being fully invisible. note that unlike RGB graphics you can not manually override the alpha value for each function call.

setColor(R, G, B, A)

Combined functionality of setColor(R, G, B) with setColor(Alpha), pretty strait forward.

setRotation(Angle, X, Y)

Sets any future 'complex' drawing bill to be rotated Angle degrees around X and Y

setRotation(Angle)

Rotates the next (and only the next) 'complex' drawing bill Angle degrees around the bills center

setRotation()

clears all rotation effects

Simple drawing bills

This section documents 'simple' drawing bills not effected by setRotation and are easy to understand

drawPixel(X, Y)

Draws a pixel at the X and Y coordinates

drawLine(x1, y1, x2, y2)

Draws a line between the two points

fill(x1, y1, x2, y2)

draws a solid color square between the two points

fill(R, G, B)

clears the screen with the provided color, if your smart you can combine this with setColor(Alpha) to make fade transitions

floadFill(X, Y, R, G, B)

Starts floadfilling from the provided point with the provided color, only filling colors exactly matching the color the fill operation started on

floadFill(X, Y, Tolerance, R, G, B)

Starts a floadFill operation with a tolerance value that defines what even of difference is allowed when floading

Complex drawing bills

This section contains more advanced and fun drawing methods

drawRectangle(x1, y1, x2, y2)

Draws a wireframe rectangle around the two points

drawCircle(x1, y1, x2, y2)

Draws a circle between the two points, can draw ellipsis

drawCircle(X, Y, Radius)

Draws a circle Radius units around X, Y

drawPolygon(X, Y, n, Radius)

Draws a polygon around the point, with n sides, with the final shape fitting into Radius/size

toRadius(n, Apothem)

Not actually a drawing function, given n amount of size and an Apothem, it will convert that into a radius, meant to enable you to use drawPolygon with a apothem instead of a radius

drawBezier(x1, y1, x2, y2, x3, y3)

Draws a smooth Bézier curve between the three points

drawSpline(X's, Y's)

Draws a mathematical spline through the given points

drawSpline(AlternatingCord's)

Draws a spline, except with the X and Y cords interweaved into one list
